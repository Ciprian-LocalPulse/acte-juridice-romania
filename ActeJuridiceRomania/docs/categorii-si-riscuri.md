# Categorii si riscuri

## Risc redus

Documente interne simple, cereri administrative generale, cereri catre angajator, procese-verbale de predare-primire.

## Risc mediu

Contracte cu bani, termene, garantii sau penalitati. Exemple: inchiriere, prestari servicii, vanzare-cumparare bun mobil.

## Risc ridicat

Documente care implica imobile, minori, mosteniri, firme cu obligatii fiscale complexe, garantii reale, executare silita, litigii, reprezentare notariala sau bancara.

## Regula practica

Cu cat miza este mai mare, cu atat modelul trebuie tratat mai mult ca lista de verificare si mai putin ca document final.
