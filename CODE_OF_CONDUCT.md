# Cod de conduita

Acest proiect este pentru ajutor public, claritate si acces la informatie. Discutiile trebuie sa ramana respectuoase, utile si orientate spre imbunatatirea documentelor.

Nu sunt acceptate:

- atacuri personale;
- publicarea datelor personale;
- consultanta juridica personalizata prezentata ca certitudine;
- continut discriminatoriu;
- spam sau promovare agresiva.

Pentru probleme sensibile, deschide un issue fara date personale sau contacteaza un profesionist.
