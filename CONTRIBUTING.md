# Contribuie

Multumim ca vrei sa imbunatatesti proiectul.

## Ce acceptam

- modele noi de documente scrise clar;
- corecturi de limba si structura;
- surse oficiale mai bune;
- checklist-uri mai utile;
- exemple de completare anonimizate;
- imbunatatiri de accesibilitate pentru README si documentatie.

## Ce nu acceptam

- documente copiate din surse platite sau proprietare;
- promisiuni de tipul "garantat valabil";
- consultanta juridica personalizata in issue-uri;
- date personale reale;
- clauze abuzive sau periculoase fara avertisment.

## Format pentru un model nou

Fiecare model trebuie sa includa:

- titlu;
- cand se foloseste;
- cand nu se foloseste;
- textul modelului;
- checklist inainte de semnare;
- observatii privind notarul, avocatul, ANAF, ITM sau alta institutie, daca este cazul.

## Pull request

In descrierea PR-ului include:

- categoria documentului;
- problema rezolvata;
- sursele consultate;
- data verificarii;
- riscuri sau limitari.
