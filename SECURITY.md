# Security

Acest repository nu proceseaza date personale si nu ruleaza o aplicatie. Totusi, modelele pot incuraja accidental includerea unor date sensibile.

## Raporteaza responsabil

Daca observi:

- date personale reale intr-un document;
- linkuri malitioase;
- instructiuni periculoase;
- continut copiat din surse neautorizate;

deschide un issue sau un pull request care elimina problema.

Nu publica CNP-uri, serii de acte, adrese reale, numere de telefon sau conturi bancare ale altor persoane.
