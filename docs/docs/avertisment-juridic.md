# Avertisment juridic

Acest repository contine modele generale de documente si materiale educationale. Ele nu reprezinta consultanta juridica, fiscala, contabila sau notariala.

Folosirea unui model se face pe propria raspundere. Inainte de semnare, verifica:

- identitatea partilor;
- capacitatea partilor de a semna;
- obiectul documentului;
- obligatiile, termenele si sumele;
- forma ceruta de lege;
- daca documentul trebuie autentificat, inregistrat sau depus la o institutie.

Consulta un profesionist pentru:

- tranzactii imobiliare;
- sume importante;
- garantii, penalitati sau clauze complexe;
- mosteniri, divort, minori sau tutela;
- relatii de munca sensibile;
- litigii;
- documente fiscale sau contabile cu impact major.

Modelele trebuie adaptate la situatia concreta si la legislatia in vigoare la data folosirii.
