# Ghid de adaptare a unui model

## 1. Identifica scopul

Scrie intr-o propozitie ce vrei sa obtina documentul. Daca scopul nu este clar, modelul ales probabil nu este potrivit.

## 2. Completeaza partile

Verifica numele, CNP/CUI, adresa, seria actului de identitate, calitatea persoanei care semneaza si datele de contact.

## 3. Descrie obiectul

Obiectul trebuie sa fie concret: bunul, serviciul, locuinta, suma, perioada, termenul sau obligatia.

## 4. Verifica obligatiile

Pentru fiecare parte, noteaza:

- ce trebuie sa faca;
- pana cand;
- ce se intampla daca nu face;
- ce documente dovedesc executarea.

## 5. Verifica forma

Unele acte pot necesita forma autentica notariala, inregistrare fiscala, comunicare catre institutie sau semnatura electronica.

## 6. Pastreaza dovezi

Pastreaza copii semnate, confirmari de primire, chitante, transferuri bancare, e-mailuri si orice anexa mentionata in document.
