# Verificari manuale inainte de publicare

Ruleaza aceste verificari cand adaugi sau modifici documente:

## Continut

- Documentul are scop clar.
- Campurile editabile sunt marcate cu `[PARANTEZE]`.
- Exista checklist.
- Exista avertisment unde documentul poate necesita notar, avocat, ANAF, ITM sau alta institutie.

## Siguranta

- Nu exista date personale reale.
- Nu exista linkuri suspecte.
- Nu exista continut copiat din surse platite sau proprietare.
- Nu sunt promisiuni de tipul "garantat valabil".

## Surse

- Sursele oficiale sunt mentionate in PR.
- Data verificarii este trecuta in descrierea PR-ului.
- Linkurile importante se deschid corect.
